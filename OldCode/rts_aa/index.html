RTS
This is the pre-alpha version of the RTS project. Lots of features are still yet to be implemented.
To play RTS:
  • JavaScript must be enabled.
  • Make sure your browser can handle HTML5.
  • Select a building from the build menu by either clicking on the button or pressing the number displayed.
  • Hovering over the buildings in the build menu provides you with information regarding that building.
  • Build buildings by clicking somewhere on the buildable area of the screen. buildings cannot overlap!
  • Press H to hide various aspects of the user interface. this hotkey may be changed in the future.
  • Press escape to return to the main menu.
